# Project Setup
    
    To run this project, follow these steps:
    
    1. Extract the zip file.
    2. Run `npm install` to install dependencies.
    3. Run `npm run dev` to start the development server.
    
# Interactive Product Box UI

This project is part of a Frontend Technical Support screening task.  
The requirement was to build three clickable product boxes that expand and show extra options such as colors and sizes. The design and layout were provided in a Figma file, and the expand behavior was shown through a sample video.

# What I Built

- A layout containing three product boxes (top, middle, bottom)
- Each box expands when clicked
- Expanded boxes display color and size options
- No external libraries were used; everything is done using HTML, CSS and vanilla JavaScript
- The user interface closely follows the design provided in the Figma file

---

# Main Features
- Expand and collapse animation on click  
- Selectable color options  
- Selectable size buttons  
- Responsive design suitable for different screen sizes
